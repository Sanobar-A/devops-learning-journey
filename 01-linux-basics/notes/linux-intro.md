\# Linux Basics – Day 1



\## What is Linux?

Linux is an open-source operating system widely used in servers, cloud platforms,

containers, and DevOps environments.



\## Why DevOps Engineers use Linux

\- Most production servers run Linux

\- Better performance \& stability

\- Powerful CLI \& automation

\- Native support for DevOps tools



\## Common Linux Distributions

\- Ubuntu

\- CentOS / Rocky Linux

\- Amazon Linux



\## Filesystem Basics

\- / – root directory

\- /home – user files

\- /etc – configuration files

\- /var – logs \& variable data
